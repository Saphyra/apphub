Konzol

Adatot legegszerűbben a console.log(adat); metódus hívásával tudunk adatot megjeleníteni.
Na de mi is az a konzol? Ez elsősorban egy szöveges kimenet, ahova a program fejlesztők olyan adatokat írnak ki, amik segítségével nyomon tudják követni a program futását, de ezek az adatok a felhasználó számára értéktelenek, és zavaróak.

document.write


            
A konzol használatával saját magunkkal tudunk, a felhasználó számára alap esetben láthatatlan adatokat közölni. Ha azonban az adatokat a felhasználóval is meg szeretnénk osztani, akkor módosítanunk kell a HTML-t a JavaScript segítségével.

            
Erre egy lehetőség a document.write(adat); funkció használata. Ez a metódus közvetlenül a <BODY>-ba illeszti a paraméterben kapott adatot.


            

                Példa:
                
    
                    
  1. 
                        Másoljuk a következő kódpt a head.js fájlunkba:
                        
    
                    
    2. 
                    
  2. Nyissuk meg az index.html fájlunkat a böngészőben
    3. 
                


                Eredmény:
                
            


            

                Megjegyzés: Ha a document.write parancsot az után futtatjuk le, hogy a HTML teljesen betöltődött (például egy gombnyomás hatására), akkor a parancs kitörli a
                <BODY> tartalmát, mielőtt bele írná az általunk megadott paramétert. Ez a mellékhatás miatt a document.write metódus használata nem javasolt.

                Eredmény:
                
            


            

                Megjegyzés: Amíg a felugró ablak aktív, addig blokkolja az oldal futását. Mint a példán is látható, még a HTML tartalma sem töltődött be, és nem is fog, amíg a felugró ablak nincsen "leokézva". Így a felhasználót rá kényszerítjük arra, hogy az általunk közölni kívánt infirmációval foglalkozzon, aminek ő nem biztos, hogy örül. Ezért az
                alert-et csak indokolt esetben használjuk.
            

        


        

            
innerHTML


            
A node.innerHTML = adat; segítségével már meglévő HTML elemektartalmát tudjuk módosítani.


            

                Példa:
                
    
                    
  1. 
                        Másoljuk a következő kódpt a body.js fájlunkba:
                        
    
                    
    2. 
                    
  2. Nyissuk meg az index.html fájlunkat a böngészőben

                Eredmény:
                
                Magyarázat: a document.getElementById(id); metódus segítségével a HTML megadott azonosítójú tagjét (node) kapjuk meg. Ennek a tagnek a
                innerHTML tulajdonságát megadva tudjuk módosítani a weboldalon megjelenő tartalmat.
            


            
Ezzel a megoldással a JavaScript segítségével át tudjuk formálni az egész weboldalunkat, vagy csak egy részét, annak újra töltése nélkül.


            
Megjegyzés: A
                document.getElementById(id); csak a már betöltődött HTML-ben keres, így az ezt használó JavaScript kódot csak a HTML renderelése után lehet használni.
            

        


        

            
innerText


            

                A node.innerText = adat; utasítás ugyanúgy működik, mint a node.innerHTML, azzal a különbséggel, hogy ha az adat esetleg más HTML tageket használ (mint a
                <B> tag az előbbi példából), akkor azt nem rendereli ki HTML-ként, hanem ugyanúgy szövegként jeleníti meg.
