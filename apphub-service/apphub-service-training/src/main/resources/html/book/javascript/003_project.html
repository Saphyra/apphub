JavaScript - Projekt

Mielőtt tovább mennénk, állítsunk össze egy kis weboldalt, amin keresztül ki tudjuk próbálni a különböző kódokat.
  1. Hozzunk létre egy mappát, amibe a weboldalunk féjljai kerülnek!
  2. Hozzunk létre egy fájlt style.css néven! (Egyelőre hagyjuk üresen)
  3. Hozzunk létre egy fájlt head.js néven! (Egyelőre hagyjuk üresen)
  4. Hozzunk létre egy fájlt body.js néven! (Egyelőre hagyjuk üresen)
  5. Hozzunk létre egy fájlt index.html néven, és másoljuk bele a következő HTML kódot: 
    
                    
    6. 
                

            

        


        

            
A HTML a következő dolgokat tartalmazza:


            

                
    
                    
  • Egy <TITLE> taget, az oldal címének megadásához
    • 
                    
  • A <LINK> tag segítségével be importálja a még üres style.css fájlt
    • 
                    
  • A <SCRIPT src="/head.js"></SCRIPT> be importálja a még üres head.js fájlt
    • 
                    
  • 
                        Egy kis HTML struktúrát, amiből az eredmeny azonosítóval ellátott
                        <SPAN> a fontos számunkra, mert ebbe a tagbe fogjuk írni a JavaScript által kiszámított eredményünket. (Arról majd később, hogy pontosan hogyan.)
                    
    • 
                    
  • A <SCRIPT src="/body.js"></SCRIPT> be importálja a még üres body.js fájlt
    • 
                

            


            
Van egy <SCRIPT> tagünk a <HEAD>-ben és a
                <BODY>-ban is. Így egyszerre tudunk a weboldal tartalma előtt, és után is futtatni JavaScript kódot, ami segíti majd a szemléltetést.
            

        


        

            
Érdemes az így létrehozott weboldalról minden példa kipróbálása előtt másolatot készíteni, így az egyes példákhoz tartozó kód külön-külön is megtekinthető lesz, valamint nem kell minden egyes alkalommal vissza állítani a weboldalunkat az eredeti állapotába.
