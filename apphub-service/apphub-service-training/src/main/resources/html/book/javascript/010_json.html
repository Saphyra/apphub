JavaScript - JSON

Mi az a JSON?

Eddig a könyvben arra fókuszáltunk, hogy hogyan tudjuk a JavaScript segítségével feldolgozni és megjeleníteni a különböző adatokat a felhasználó böngészőjében. Viszont mi van akkor, ha a JavaScript által kiszámított adatokat egy másik rendszerbe szeretnénk átvezetni, vagy az adatok, amikkel dolgozni szeretnénk egy másik rendszerben vannak elhelyezve (például kliens-szerver kommunikáció)? Ahhoz hogy két rendszer kommunikálni tudjon egymással, közös nyelvre van szükség. Csinálhatunk az adatainknak saját reprezentációt, vagy használhatunk egy már meglévő formátumot.
Ilyen formátum például az XML, a CSV, vagy a JSON.

De hogyan néz ki?

Ahhoz, hogy a kliens és a szerver kommunikálni tudjon egymással, előbb a memóriából egy továbbítható formátummá (szöveg, bájtkód, stb) kell alakítani az adatot, majd ezt a formátumot beolvasni a címzett memóriájába.
A JSON az alapvető adatstruktúrák szöveges megfelelője, ennek megfelelően léteznek benne primitív adattípusok (number (szám), string (szöveg), boolean (igaz/hamis)), valamint tömbök és objektumok is.
