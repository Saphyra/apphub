Változók létrehozása


            
Változókat a var, let és const kulcsszavakkal tudunk létre hozni.
            


            

                
    
                    
  • 
                        var: var szoveg = "alma"; - Elavult, nem használják, egyes esetelben használata "furcsa problémákhoz" vezethet.
                    
    • 
                    
  • 
                        const: const szoveg = "alma"; - Írásvédett változó létrehozása. Értékét a létrehozásakor meg kel adni, és utána nem módosítható.

                        let: let szoveg = "alma" - Többször írható változó létrehozása.

                A nullhoz hasonló érték az undefined, ami két esetben fordul elő:
                
    
                    
  • 
                        A változónkat létre hoztuk, de még nem kapott értéket:
                        
    
                    
    • 
                    
  • 
                        A funkciók alapértelmezett visszatérési értéke
                        
    
                    
    •

Funkciók


            

                Funkció létrehozásának hátom módja:
                
    
                    
  • function funkcioNev(parameter1, parameter2){} - Létrehozzuk "funkcioNev" funkciót, mely két paramétert vár ("parameter1" és "parameter2).
    • 
                    
  • const funkcioNev = function(parameter1, parameter2){} - Létrehozzuk "funkcioNev" változót, aminek értékül adjuk az utána létre hozott névtelen funkciót.
    • 
                    
  • const funkcioNev = (param) => {} - Úgynevezett "nyíl" (arrow) funkció. Zárójelben a paraméter lista, majd egy "nyíl", végül a funkció törzse.
    • 
                

                A fenti három megoldás használatában nem tér el egymástól.
            


            
Funkció meghívása: const visszateresiErtek = funkcioNev(param);

            

                Megjegyzés: Hiába vár egy funkció paraméter(eke)t, azokat nem kötelező kitölteni. Ebben az esetben a funkción belül a meg nem adott paraméterek értéke
                undefined lesz. Ezt a jelenséget kihasználhatjuk opcionális metódus paraméterek létrehozására is.
