CSS - Átmenet

Mi is ez?

Korábbi fejezetekben megtanultuk, hogy a CSS segítségével bizonyos felhasználói beavatkozás hatására megváltozhatnak az elemek tulajdonságai (:hover kiválasztó például).
A transition tulajdonság abban segít, hogy ezek a változások ne ugrásszerűen, hanem fokozatosan jussanak érvényre.
Ez a tulajdonság értékül egy másik tulajdonságot vár (azt, amelyiknek a változását gördülékennyé szeretnénk tenni), valamint egy idő értéket, hogy ez az átalakulás mennyi idő alatt menjen végbe.
Például: 



                

            


            
Ahogyan a példában látható, egyszerre több tulajdonságra is lehet átmenetet beállítani, ekkor a különböző értékeket vesszővel elválasztva soroljuk fel egymás mögött.

        


        

            
Átmenet sebességének a módja


            
A transition-timing-function azt a metódust adja meg, hogy az átmenet sebessége az átalakulás során hogyan változzon.


            

                Lehetséges értékek:
                
    
                    
  • ease - lassú kezdés, aztán gyorsul, majd megint lassul. (alapértelmezett)
    • 
                    
  • linear - állandó sebesség
    • 
                    
  • ease-in - lassú kezdés
    • 
                    
  • ease-out - lassú befejezés
    • 
                    
  • ease-in-out - lassú kezdés és befejezés
    • 
                

            


            

                Példák:
                
linear

                
ease-in

                
ease-in-out

            

        


        

            
Átmenet késleltetése


            
A transition-delay tulajdonsággal azt adhatjuk meg, hogy az átmenet mennyivel az eseményt kiváltó ok után keződjön el.


            

                Példa: transition-delay: 2s;
                
Az átmenet csak két másodperc múlva fog elkezdődni.
