CSS - Listák

Lista stílus

A listák elemei előtt megjelenő jelet a list-style-type tulajdonsággal állíthatjuk:
    Bővített lista
    Megjegyzés: a list-style-type megváltoztathatja a lista visekledését, és rendezett listából is lehet rendezetlen, és fordítva.

    Kép beállítása a listaelemek előtt

    Erre a list-style-image tulajdonság szolgál: 
    list-style-image: url("/img/css/sample_background.jpg")

      Listaelem jelzők elhelyezkedése
