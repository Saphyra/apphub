Versenyhelyzetek


            
Egy weblap több helyen is tartalmazhat CSS kódot: tag attribútumaként, az oldal fejlécében elhelyezett blokkokban és importált külső fájlokban.

            

                Vegyünk egy egyszerű példát:

                Van egy oldalunk, aminek a háttérszínét szeretnénk módosítani. A
                <body> tag tulajdonságában megadjuk, hogy a háttér színe legyen zöld, a fejlécben azt mondjuk, hogy legyen sárga, míg a beimportált css fájl inkább kékre állítaná.

                Na de akkor mi lesz a végeredmény?
            

            

                A fentebb említett versenyhelyzetekben a következő szerint lépnek érvénybe a szabályok:
            
    
                
  1. A !important (fontosnak jelölt) szabályok
    2. 
                
  2. A tag attribútumában megadott szabályok
    3. 
                
  3. Amelyik később töltődött be (az importálás és fejlécbe helyezett blokkok sorrendje szerint a legutolsó)
    4. 
                
  4. Böngésző beépített szabályai (User Agent Stylesheet)
    5.
